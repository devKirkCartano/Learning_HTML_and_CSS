8 Important CSS Properties You Must Know

1. Display

Most Commonly Used values

  • display: block;
  • display: inline-block;
  • display: inline;
  • display: none;

2. Width and Height

Commonly used with display:block and display:inline

Common units for width and height

  • px - pixels
  • em - A unit of measurement, where 1 em = current font size.
  • rem- same as em, but makes life easier without the inheritance problem
  • % - percentage
  • auto - highly useful unit

3. Margin and Padding

Dictates the spaces between elements on your website (same unit as width and height)

Margin affects the area outside of borders

Padding affects area inside the border

4. Border

3 different properties

  • border-width - width of the border(same unit as width and height
  • border-style - style of the border (usual values are solid and dashed)
  • border-color - color of the border (Hex and rgb values can be used)

5. Floats

Positions the edge of the targeted HTML content at the edge of one side of the parent container.

3 basic properties

  • float: left;
  • float: right;
  • float: none;

6. Color

Refers to text color, it takes on a #hex values or a rgb value as with border colors.

7. Background

Background refers to the Background of the HTML element.

Some background properties:

  • background-color: color of the background, (takes #hex or rgb value
  • background-image: URL, takes on the path to your image
  • background-repeat: repeat, takes on repeat, repeat-x, repeat-y, no-repeat (it repeats the background if the width of the element is larger)
  • background-position: position of the background relative to the HTML element. (2 values needed : x: amount of offset from the left, y: amount of offset from the top

8. Font

Appearance of the text in website. Some font properties:

  • font-style: italic or normal
  • font-variant: normal or small caps
  • font-weight: normal, bold, bolder, 100-900
  • font-size: examples: 6,8,10,12,14,16,
  • line height: determines the space above and below the text
  • font-family: declare type faces and fonts to be used
